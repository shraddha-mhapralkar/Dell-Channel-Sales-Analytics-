# Dell-Channel-Sales-Analytics-
End-to-end Dell channel Sales Analytics project using Python, MySQL, Power BI, DAX, and Excel.

## Project Overview
This project analyzes Dell's channel sales data to identify sales performance, customer trends, regional performance, product demand, and operational efficiency. The project demonstrates an end-to-end data analytics workflow using Python, SQL and Power BI.

---

## Business Problem
Dell wants to understand:
- Which regions generate the highest revenue.
- Which customer segments contribute the most sales.
- Which product categories perform best.
- Which salespeople and managers achieve the highest revenue.
- How delivery performance and opportunity status impact business.
